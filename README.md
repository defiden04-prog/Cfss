# Claim Free Solana DApp

Immersive Solana account reclamation tool. Identify empty token accounts and reclaim rent-exempt SOL.

## Features
- **Account Scanning**: Matrix-themed digital rain scanner.
- **Rent Reclamation**: Batch close token accounts and refund SOL to owner.
- **Referral System**: Multi-tier commission structure (30% base).
- **Pro Automation**: Auto-sweep and monitoring features.

## Setup
1. `npm install`
2. `npm run dev`

## Deployment
Push to GitHub to trigger Vercel deployment.
